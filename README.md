# Vendor Onboarding System

A full-stack Vendor Onboarding System built using ASP.NET and SQL Server, designed to simulate real-world enterprise workflows with multi-level approvals and document management.

## 🚀 Features

- Multi-level approval workflow
- Role-based access control
- Vendor document upload & verification
- Structured database design for tracking approvals
- Scalable backend architecture
